# 🌌 Sky Events – AI Powered Event Planner

## 📌 Overview

Sky Events is an AI-powered event planning web application that helps users plan, manage, and optimize events efficiently.  
It includes multiple intelligent tools like event recommendation, budget planning, schedule generation, and more — all built using pure JavaScript.

This project demonstrates how AI-like logic can be implemented without external APIs using front-end technologies.

---

## 🚀 Features

✨ 12 AI-Based Tools:

- 🎯 Event Recommender
- ✍️ Event Description Generator
- 💰 Budget Planner
- 👥 Guest List Optimizer
- 🤖 AI Chatbot Assistant
- 📅 Event Schedule Generator
- 💡 Tagline Generator
- 📊 Feedback Analyzer
- 📧 Email Composer
- 🏟️ Venue Advisor
- 📱 Social Media Caption Generator
- 🛡️ Risk Assessment Tool

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🧠 How It Works

The application uses predefined datasets and logic-based algorithms to simulate AI behavior.  
For example:

- Event recommendations are based on keyword matching and scoring
- Budget planning uses percentage-based allocation
- Feedback analysis uses sentiment keyword detection
- Chatbot uses rule-based responses
